¡Bienvenidos a CICAPY!

En el Instituto Técnico Superior Centro de Investigación y Capacitación Paraguay (CICAPY), nos especializamos en ofrecer educación virtual de alta calidad. Nuestro enfoque está en tecnologías modernas para el análisis de datos, potenciando tus habilidades para enfrentar los desafíos del mundo profesional.

CICAPY fue creado por la Resolución 848/2022 del Ministerio de Educación y Ciencias, consolidando nuestro compromiso con la educación de excelencia en Paraguay.

Misión

Formar profesionales altamente capacitados mediante programas educativos virtuales innovadores en tecnologías de análisis de datos, promoviendo la transformación digital y la toma de decisiones basada en datos en Paraguay y más allá.

Visión

Ser reconocidos como un referente regional en educación virtual, transformando el aprendizaje en análisis de datos y contribuyendo al desarrollo tecnológico y profesional de nuestros estudiantes.

📊 Capacitación empresarial a medida: online, in-company o híbrida.

Ofrecemos programas de formación diseñados para equipos que buscan potenciar sus capacidades en Análisis de Datos e Inteligencia de Negocios mediante el uso de herramientas líderes del mercado: R, Python, Power BI, SQL y Excel. Las capacitaciones pueden realizarse 100% online o in-company, adaptadas a las necesidades de tu organización.

  • 📌 Programas personalizados según el sector y los datos reales de tu empresa.
  • 🧠 Formación práctica enfocada en análisis de datos, automatización de reportes y dashboards.
  • 🎯 Modalidad online en vivo o presencial en tu oficina (in-company).
  • 📁 Material de clase, datasets y grabaciones disponibles para los participantes.
  • ✅ Ideal para equipos de análisis, finanzas, marketing, RRHH, operaciones y más.

DAX Aplicado — Análisis de Datos con Power BI y Excel

Curso avanzado de DAX con Power BI y Excel, modalidad sincrónica y asincrónica. Participá en las clases en vivo o accedé a las grabaciones a tu propio ritmo.

✅ Metodología

  • Clases en vivo a través de Zoom.
  • Grabaciones disponibles para consulta posterior.
  • Actividades prácticas: ejercicios y proyectos aplicados.
  • Asesoramiento personalizado durante y después de las sesiones.

📊 Contenido del Curso — 4 sesiones

🧩 Sesión 1 – Modelado de Datos y Fundamentos Avanzados de DAX

Objetivo: Diseñar un modelo robusto que optimice los análisis.

  • Principios del modelado estrella y buenas prácticas.
  • Relación entre tablas (activa/inactiva) y su impacto en los cálculos.
  • Uso de USERELATIONSHIP, CROSSFILTER, TREATAS.
  • Práctica: construir un modelo de ventas y entregas.

📈 Sesión 2 – Métricas Avanzadas para Análisis de Desempeño

Objetivo: Crear KPIs avanzados que apoyen la toma de decisiones.

  • Funciones avanzadas: RANKX, DIVIDE, SUMX, VAR.
  • Medidas dinámicas: clientes TOP N, productos estrella, tasas de conversión.
  • Práctica: panel de rendimiento comercial.

📆 Sesión 3 – Inteligencia de Tiempo y Contexto en DAX

Objetivo: Aplicar análisis temporales sofisticados para comparativos y tendencias.

  • Construcción de calendarios y columnas de fecha personalizada.
  • Time Intelligence: SAMEPERIODLASTYEAR, DATESMTD, TOTALYTD, DATEADD.
  • Práctica: comparación YTD/LYTD y rolling averages.

⚙️ Sesión 4 – Casos Reales, Optimización y Buenas Prácticas

  • Análisis de desempeño con Performance Analyzer.
  • Limpieza y optimización de medidas (uso de VAR, simplificación de filtros).
  • Proyecto final: panel financiero, panel de operaciones y panel de ventas/forecast.

📁 Material Incluido

  • Archivos de práctica en Excel y Power BI.
  • Presentaciones en PDF por sesión.
  • Guía rápida de funciones DAX clave.
  • Grabación de todas las sesiones en vivo.

Python & Git & GitHub

Curso práctico de análisis de datos con Python, control de versiones con Git y publicación de proyectos con GitHub Pages, con soporte de GitHub Copilot e IA.

Módulo 1: Entorno de trabajo y GitHub Copilot

Objetivo: Dejar listo el entorno de ciencia de datos y habilitar IA asistida en el editor.

  • Instalación: Python 3.11+, VS Code, extensiones Python, Jupyter, GitHub Copilot.
  • Git + GitHub: claves, configuración de usuario, git init, add, commit, push.
  • Estructura base del repo: /datasets, /notebooks, /src, /docs.

Módulo 2: Python para análisis de datos

Objetivo: Dominar lo esencial para manipular datos con Python.

  • Fundamentos: tipos, listas/dicts, funciones, comprensión de listas.
  • pandas y numpy: carga (CSV/Excel/JSON), info(), describe().
  • Limpieza: nulos, duplicados, tipos y normalización de columnas.

Módulo 3: Visualización y dashboards interactivos

Objetivo: Comunicar insights con gráficos claros e interactivos.

  • Exploratoria: matplotlib, seaborn (barras, líneas, boxplot, heatmap).
  • Interactividad: plotly.express.
  • Mapas: folium (capas y marcadores).

Módulo 4: Git y colaboración

Objetivo: Adoptar flujo colaborativo profesional.

  • Branching: feature/, fix/, release/.
  • Pull Requests: descripción, checklist, code review.
  • Copilot Chat: revisión de diffs y sugerencias de refactor.

Módulo 5: Publicación web con GitHub Pages

Objetivo: Publicar análisis y resultados en una web navegable.

  • Exportar notebooks a HTML con nbconvert.
  • Activar Pages: branch main + carpeta /docs.
  • Automatización opcional con GitHub Actions.

Módulo 6: Proyecto integrador

Objetivo: Construir un mini-portafolio con un caso completo.

  • Dataset a elección: ventas, Netflix, clima, fútbol, e-commerce.
  • Pipeline: ingesta → limpieza → EDA → visualizaciones → conclusiones.
  • Entrega: repo ordenado + página en GitHub Pages.

Módulo 7: Modelos estadísticos e IA aplicada

Objetivo: Incorporar predicciones y técnicas de IA al flujo analítico.

  • Estadística aplicada: correlaciones, pruebas simples, PCA.
  • Modelos: regresión lineal y logística, árboles con scikit-learn.
  • Métricas: RMSE, accuracy, F1, matriz de confusión.
  • Copilot/ChatGPT para generar, explicar y documentar pipelines.
